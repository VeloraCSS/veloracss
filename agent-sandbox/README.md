# Agent Sandbox

This directory is for testing, prototyping, and debugging agent workflows, patches, and automation scripts in isolation from the main VeloraCSS codebase.

## Usage
- Place test scripts, agent instructions, and workflow prototypes here.
- Use for rapid iteration and validation before applying changes to production code.

## Structure
- /scripts: automation and patch scripts
- /tests: workflow and patch test cases
- /instructions: agent prompt and skill files

## Guidelines
- Keep sandbox experiments isolated.
- Clean up obsolete files regularly.
- Document any reusable patterns or lessons learned.
