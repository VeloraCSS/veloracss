# VeloraCSS Monorepo

VeloraCSS is a modular CSS framework and ecosystem, organized as a monorepo for easy development, integration, and deployment.

## Structure

This repository contains:

- **packages/veloracss/** — Core VeloraCSS framework source
- **packages/tracker/** — Project tracker and Discord integration
- **packages/brand/** — Design assets and branding
- **packages/docs/** — Documentation site and guides

## Getting Started

1. **Install dependencies**
   - Use `npm install` or your preferred workspace tool (pnpm, Yarn).

2. **Develop**
   - Work on any package in `packages/`.
   - Each package has its own README and scripts.

3. **Build & Test**
   - Run build/test scripts from the root or individual package folders.

## Monorepo Management

- Use workspace tools (npm workspaces, pnpm, Yarn) for cross-package development.
- All packages share a single git history.

## Contributing

- Fork the repo and submit PRs for any package.
- See individual package READMEs for details.
