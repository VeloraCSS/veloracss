# VeloraCSS Docs

This folder is for the documentation site and guides for VeloraCSS.

## Setup Steps

1. Choose a static site generator or framework (e.g. SvelteKit, VitePress, Next.js, Astro).
2. Initialize the docs site in this folder.
3. Add core documentation pages (index, guides, API, examples).
4. Integrate with monorepo for live code and style references.
5. Add build and dev scripts to package.json.
6. Test docs site locally and deploy as needed.

## Example Structure

- docs/
  - index.html (or src/pages/index.svelte, etc.)
  - guides/
  - api/
  - examples/
