# API Reference

## Tokens

VeloraCSS uses design tokens for color, spacing, typography, and more.

```css
:root {
  --vel-space-1: 0.25rem;
  --vel-space-2: 0.5rem;
  --vel-space-4: 1rem;
  --vel-space-5: 1.5rem;
  --vel-space-8: 3rem;
}
```

## Utilities

Use utility classes for layout, flex, grid, and more.

```html
<div class="vel-row-between">
  <div>Left</div>
  <div>Right</div>
</div>
```
